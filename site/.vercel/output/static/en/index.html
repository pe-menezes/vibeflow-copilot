Stop fixing what AI built wrong

Vibeflow is a methodology that makes AI code exactly what you specified — nothing more, nothing less.

Define the scope before coding starts. The AI receives a closed brief grounded in your real codebase and implements following your patterns. Then you audit the result.

Use cases

Vibeflow adapts to different roles and scenarios.

Developer

New feature in an existing project

You need to add a payment flow. Instead of letting AI guess, you define scope (Stripe only, no PayPal), anti-scope (no subscription management), and a 5-check DoD. The agent implements exactly that.

discover → gen-spec → prompt-pack → implement → audit
Developer

Quick bug fix with controlled scope

A date formatting bug in the dashboard. You use quick mode — it generates a focused prompt pack in seconds. The agent fixes the bug, nothing else. Audit confirms the fix.

quick → implement → audit
Product Manager

Defining requirements before handing off

You have a feature idea but need to clarify scope before the dev team starts. Discover walks you through the hard questions and produces a clear PRD. No code involved.

discover → hand off PRD to team
Tech Lead

Onboarding a new developer

New dev joins the team. Instead of reading 200 files, they run analyze — Vibeflow produces a comprehensive guide to the project: architecture, patterns, conventions, all with real code examples.

analyze → dev reads .vibeflow/

Who is it for

Developers using AI agents

You use Copilot, Claude Code, or Cursor daily. You want AI to follow your patterns, respect scope, and produce auditable results.

Tech leads who want consistency

Your team uses AI but the output quality varies wildly. Vibeflow standardizes the process: spec first, implement second, audit always.

PMs who want to define the "what"

You want to clarify requirements and scope before developers start coding. Discover and gen-spec let you do that without writing a single line of code.

What's included

Every command has a clear purpose.

discover

Interactive dialogue that turns a vague idea into a clear PRD. Challenges assumptions, cuts scope, forces decisions.

For anyone with an idea
analyze

Deep-scans your codebase and produces documentation: stack, architecture, patterns, conventions — all with real code examples.

For project setup
gen-spec

Generates a technical specification with a binary Definition of Done (3-7 pass/fail checks), scope, anti-scope, and risks.

For clear requirements
prompt-pack

Creates a self-contained prompt for the coding agent. Embeds your real patterns and conventions so the AI follows them.

For consistent implementation
audit

Verifies the implementation against the spec. Checks DoD, patterns, and tests. Produces a PASS, PARTIAL, or FAIL report.

For quality verification
quick

Fast-track for small tasks. Skips discovery, generates a prompt pack directly. For bug fixes and tasks under 4 files.
