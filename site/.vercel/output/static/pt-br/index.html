Pare de consertar o que a IA construiu errado

Vibeflow é uma metodologia que faz a IA codar exatamente o que você especificou — nada mais, nada menos.

Defina o escopo antes de começar a codar. A IA recebe um brief fechado baseado no seu codebase real e implementa seguindo seus padrões. Depois você audita o resultado.

Já aconteceu com você?

Esses são os custos reais de codar com IA sem estrutura.

IA altera 47 arquivos pra uma feature simples

Você pediu uma página de login. A IA adicionou OAuth, reset de senha, verificação de email, gerenciamento de sessão — coisas que você nunca pediu.

Ela ignora os padrões do seu projeto

Seu projeto usa uma estrutura de pastas específica, convenções de nomenclatura e padrões arquiteturais. A IA não sabe nada disso e cria código que parece de outro projeto.

Sem forma de verificar o resultado

A IA diz "pronto!" mas você não tem checklist, nenhum critério, nenhuma forma objetiva de validar se o que foi construído corresponde ao que foi pedido.

Horas debugando código gerado por IA

Ao invés de economizar tempo, você gasta horas entendendo e consertando o que a IA produziu. A "velocidade da IA" desaparece.

Casos de uso

Vibeflow se adapta a diferentes papéis e cenários.

Desenvolvedor

Feature nova em projeto existente

Você precisa adicionar um fluxo de pagamento. Ao invés de deixar a IA adivinhar, você define escopo (só Stripe, sem PayPal), anti-escopo (sem gerenciamento de assinatura), e um DoD com 5 checks. O agent implementa exatamente isso.

discover → gen-spec → prompt-pack → implement → audit
Desenvolvedor

Bug fix rápido com escopo controlado

Um bug de formatação de data no dashboard. Você usa o modo quick — gera um prompt pack focado em segundos. O agent corrige o bug, nada mais. Audit confirma o fix.

quick → implement → audit
Product Manager

Definir requisitos antes de passar pro time

Você tem uma ideia de feature mas precisa clarificar escopo antes do time de dev começar. O discover te guia pelas perguntas difíceis e produz um PRD claro. Sem código envolvido.

discover → entregar PRD ao time
Tech Lead

Onboarding de dev novo

Dev novo entra no time. Ao invés de ler 200 arquivos, roda analyze — o Vibeflow produz um guia completo do projeto: arquitetura, padrões, convenções, tudo com exemplos reais de código.

analyze → dev lê .vibeflow/

Pra quem é

Devs que usam agentes de IA

Você usa Copilot, Claude Code ou Cursor diariamente. Quer que a IA siga seus padrões, respeite escopo e produza resultados auditáveis.

Tech leads que querem consistência

Seu time usa IA mas a qualidade do output varia muito. Vibeflow padroniza o processo: spec primeiro, implementa depois, audita sempre.

PMs que querem definir o "o quê"

Você quer clarificar requisitos e escopo antes dos devs começarem a codar. Discover e gen-spec permitem isso sem escrever uma linha de código.

O que está incluído

Cada comando tem um propósito claro.

discover

Diálogo interativo que transforma ideia vaga em PRD claro. Desafia premissas, corta escopo, força decisões.

Pra quem tem uma ideia
analyze

Deep-scan do codebase que produz documentação: stack, arquitetura, padrões, convenções — tudo com exemplos reais de código.

Para setup do projeto
gen-spec

Gera especificação técnica com Definition of Done binário (3-7 checks pass/fail), escopo, anti-escopo e riscos.

Para requisitos claros
prompt-pack

Cria um prompt auto-contido para o coding agent. Embute seus padrões e convenções reais para a IA seguir.

Para implementação consistente
audit

Verifica a implementação contra a spec. Checa DoD, padrões e testes. Produz relatório PASS, PARTIAL ou FAIL.

Para verificação de qualidade
quick

Fast-track para tasks pequenas. Pula discovery, gera prompt pack direto. Para bug fixes e tasks de até 4 arquivos.
